# NYT Best Sellers Webscraper

Webscraper que recopila 3 años de datos semanales de los libros más vendidos del NYT utilizando BeautifulSoup y requests, con hashing MD5 para eliminar duplicados de libros a través de más de 150 capturas semanales. Posteriormente carga los datos en un star schema en SQLite y los conecta a Power BI, permitiendo el análisis de la trayectoria de rankings, tendencias de frecuencia de autores y visualizaciones de la permanencia en la lista.

### Tecnologías

- Python

- SQLite

- Power BI

- BeautifulSoup

- pandas

- requests
